#  FastAPI Real-Time Chat Application

This is a real-time multi-user chat application built using FastAPI and WebSockets. It allows users to send and receive messages instantly without refreshing the page.

##  Features
- Real-time messaging using WebSockets
- Multi-user chat support
- Fast and lightweight backend with FastAPI
- Simple frontend using HTML, CSS, and JavaScript

##  Tech Stack
- FastAPI
- WebSockets
- Python
- HTML / CSS / JavaScript
